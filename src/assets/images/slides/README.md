# Sample Images Information

This folder should contain the hero slider images for your homepage.

## Required Images

1. **electrical-hero-1.jpg** - Main electrical services banner
2. **smart-home-2.jpg** - Home automation and smart solutions
3. **industrial-3.jpg** - Industrial electrical solutions
4. **solar-4.jpg** - Solar panel and renewable energy solutions

## Image Specifications

- **Recommended Size**: 1920x1080px or higher
- **Format**: JPG, PNG, or WebP
- **Quality**: High quality, professional images
- **Aspect Ratio**: 16:9

## Image Sources

You can find quality images from:
- Unsplash (unsplash.com) - Free high resolution images
- Pexels (pexels.com) - Free stock photos
- Pixabay (pixabay.com) - Free images and videos
- Shutterstock - Professional stock photography

## Optimization

Before uploading, optimize your images:
1. Compress using TinyPNG.com or similar
2. Resize to recommended dimensions
3. Convert to WebP format if possible
4. Keep file size under 500KB per image

## Implementation

Once images are added, they will automatically appear in the hero slider carousel with automatic rotation every 5 seconds.

### To update slide images:
1. Add new images to this folder
2. Update image file names in `src/app/components/hero-slider/hero-slider.component.ts`
3. Adjust slide content (title, subtitle) as needed
